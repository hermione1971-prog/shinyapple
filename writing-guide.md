## 1. 海外リモートワーク体験記事の抑えどころ

### ⚠️ 最重要：NDA（守秘義務）に注意

OutlierやDataAnnotation等のプラットフォームは、**案件の具体的な内容・クライアント名・テストの内容**の公開を契約で禁じています。

- ✅ 書いてOK：プラットフォーム名、登録手順、報酬の仕組み・金額、自分の感想、トラブル体験
- ❌ 書いてNG：案件の中身、テスト問題の内容、クライアント名、作業画面のスクリーンショット

違反するとアカウント停止のリスクがあります。迷ったら書かない。

### 💎 数字こそ財産

「10,848円が楽天銀行に振り込まれた」のような具体性が最大の差別化。記録しておくもの：

- 報酬額（ドル建て・円換算の両方）
- 時給換算・作業時間
- 振込手数料・為替レート
- 日付（登録日・合格日・初稼働日・振込日）

### 🔑 1記事1キーワード

想定検索ワードごとに記事を分ける。全部入りの長文より、悩み別の記事が検索に強い。

例：
- 「データアノテーション 在宅 未経験」
- 「AIトレーナー 50代」「AIトレーナー 主婦」
- 「海外リモートワーク 英語できない」
- 「失業給付 在宅ワーク 申告」←一次情報がほぼ無い空白地帯！

### 🏛️ 記事が5本貯まったら「まとめ記事」

「50代・未経験からの海外リモートワーク完全ガイド」のような親記事（ピラーページ）を作り、各記事へリンクする構成が王道。

### 🌟 同世代向けは空白地帯

この分野の発信者は20〜30代ばかり。「スキルなし・英語苦手・50代・介護あり」の目線で書ける人はほぼいない。**弱みに見える属性が、そのまま最強の差別化になる。**

### 📋 税金ネタは需要大、ただし慎重に

業務委託報酬の確定申告・扶養への影響は読まれるテーマ。ただしお金や健康の話（YMYL領域）は誤情報に厳しいので：

- 国税庁等の公式リンクを添える
- 「詳細は税務署・税理士にご確認ください」の一文を必ず入れる

### 😅 失敗談を恐れず書く

ポイ活記事のように、検索者は「失敗回避情報」を探している。トラブル・遠回り・恥ずかしい話ほど価値がある。

## 2. タイトルの型（SEO）

良いお手本：「家事嫌いの主婦がAudibleで人生変わった話【57歳・体験レビュー】」

- **検索されそうな言葉**をタイトルに入れる（サービス名・悩みの言葉）
- **数字**を入れる（「初報酬1万円」「26年目」「月6,325円に」）
- **【】で属性や記事タイプ**を補足（【体験談】【50代】【失敗談】）
- 弱い例：「海外リモートワークと初めての支払い」
- 強い例：「【体験談】50代主婦がAIトレーナーで初報酬1万円！未経験から海外リモートワークを始めた全記録」

## 3. 公開前チェックリスト

記事を公開する前に、上から順に確認：

- [ ] **PR表記**：アフィリエイトリンクがある記事は、冒頭に「※ 本記事にはアフィリエイト広告が含まれています。」（ステマ規制対応・必須）
- [ ] タイトルに検索キーワードが入っているか
- [ ] meta description（120字前後）を書いたか
- [ ] 見出し（h2）で構成されているか／冒頭に「この記事でわかること」
- [ ] **本文中の内部リンク**を2〜3本入れたか（関連記事カードとは別に）
- [ ] 関連記事カード（記事末尾）を設定したか
- [ ] アフィリエイトリンクに `rel="nofollow"` または `rel="sponsored"` が付いているか
- [ ] 英語版にはアフィリエイトリンクを入れない（サイトのルール）

### 公開作業（サイト更新）

- [ ] index.html の記事グリッドにカードを先頭追加
- [ ] **全ページ**のサイドバー「新着記事」を最新3件に更新（リンク付き形式で）
- [ ] **sitemap.xml に新記事を追加**（忘れやすい！）
- [ ] CLAUDE.md の記事一覧を更新
- [ ] git push ＋ wrangler デプロイ

## 4. アフィリエイトの基本ルール

- もしも・A8等の提携前に記事を書くのはOK。ただしリンクはプレースホルダーにしておき、承認後に差し込む
- 広告主の「記事作成時チェックポイント」資料は必ず読む。**資料の文面をそのまま記事にコピペしない**（読者向けにリライトする）
- 本人申込NG（自分のリンクから自分で申し込まない）案件に注意
- 手数料・料金など変わりやすい情報は「（2026年6月時点）」と時点を書くか、公式サイトへのリンクで済ませる
- 紹介するのは実際に使った・検討したものだけ（プライバシーポリシーにも明記済み）
