# 新しいセッション開始用プロンプト

新しいセッションを始めるときに、以下のプロンプトをコピーしてClaudeに送ってください。
（★の部分だけ書き換えてください）

---

## ▼ 新しい記事をアップするとき

```
ぴかりんごのブログの記事をアップしたいです。
プロジェクトの詳細はC:\Users\hermi\shinyapple\CLAUDE.md を読んでください。

記事タイトル：★タイトルをここに書く★
カテゴリ：★日常 / おしごと / 家計・リベ活 / 日本の文化★

以下が記事の本文です：

★ここに記事の本文を貼り付ける★
```

---

## ▼ ブログの修正・変更をしたいとき

```
ぴかりんごのブログを修正したいです。
プロジェクトの詳細はC:\Users\hermi\shinyapple\CLAUDE.md を読んでください。

やりたいこと：★やりたいことをここに書く★
```

---

## ▼ 使い方のヒント

- 記事の本文は日本語だけでOK。英語版はClaudeが作ってくれます。
- タイトルは日本語で伝えれば大丈夫です。
- 記事のカテゴリが分からなければ、内容を説明してClaudeに選んでもらいましょう。
- デプロイまでお願いすると、サイトへの反映もやってくれます。
